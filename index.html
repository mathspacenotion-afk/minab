미납자 예약문자 발송

에어테이블 「수납관리」에서 이번 달 납부 기록이 없는 학생(최근 납부일이 지난달 이전이거나 빈칸)을 불러와 알리고(Aligo)로 예약문자를 보냅니다.

1 설정

처음 사용 시 준비할 것 (열어보기)
① 에어테이블 토큰 — airtable.com/create/tokens에서 토큰 생성 → Scopes에 data.records:read 추가 → Access에 수납관리 베이스 추가 → 생성된 pat… 토큰을 위에 붙여넣기.

② 알리고 발송 IP 등록 — 알리고는 등록된 IP에서만 발송됩니다. 알리고 관리자 → 문자 API → 인증관리에서 이 컴퓨터의 IP를 등록하세요. (IP 확인: findip.kr)

③ 발신번호 — 알리고에 사전 등록된 발신번호만 사용 가능합니다.

설정은 이 컴퓨터 브라우저에만 저장되며 외부로 전송되지 않습니다 (알리고/에어테이블 호출 시에만 사용).

2 미납자 불러오기

이름학년/반수신번호원비납부 기준일최근 납부일상태

3 메시지 작성

아래 항목을 클릭하면 커서 위치에 삽입되며, 발송 시 학생별로 자동 치환됩니다.
[이름] [학년] [반] [원비] [납부기준일] [최근납부일] [이번달]
학생을 불러오면 미리보기가 표시됩니다.

4 예약 발송

브라우저 보안 정책상 알리고의 응답 내용은 이 페이지에서 직접 확인할 수 없습니다. 실제 접수 여부는 알리고 관리자 → 전송결과/예약전송 관리에서 꼭 확인해 주세요. 예약 취소도 그곳에서 할 수 있습니다. 접수가 안 되어 있다면 API Key · 발신번호 · 발송 IP 등록을 다시 확인하세요.
설정이 맞는지 테스트해 보기
본인 번호로 문자가 도착하면 알리고 설정(키·발신번호·IP)이 정상입니다. 건당 요금이 차감됩니다.
